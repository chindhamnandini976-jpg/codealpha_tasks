# Automatic Light Control Using LDR Sensor

## Project Overview

This project demonstrates an Automatic Light Control System using an LDR (Light Dependent Resistor) and Arduino Uno. The system automatically controls an LED based on the surrounding light intensity.

## Objective

To design and simulate a system that turns an LED ON in low-light conditions and OFF in bright-light conditions using an LDR sensor.

## Components Used

* Arduino Uno
* LDR (Photoresistor)
* LED
* 10kΩ Resistor
* 220Ω Resistor
* Breadboard
* Jumper Wires

## Software Used

* Tinkercad Circuits
* Arduino IDE

## Working Principle

1. The LDR senses the surrounding light intensity.
2. Arduino reads the sensor value through the analog input pin.
3. When the light level decreases, the sensor value becomes lower.
4. Arduino turns ON the LED automatically.
5. When the light level increases, the LED turns OFF.

## Features

* Automatic light detection
* Energy-efficient lighting control
* Real-time response to environmental light changes
* Easy to implement and cost-effective

## Applications

* Street lighting systems
* Smart homes
* Automatic room lighting
* Energy-saving lighting solutions

## Result

The Automatic Light Control System was successfully simulated using Tinkercad. The LED turned ON under dark conditions and OFF under bright conditions, achieving the desired objective.
